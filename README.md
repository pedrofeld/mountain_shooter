# Mountain Shooter

## About

Mountain Shooter is a 2D arcade game, developed with **Python** and **Pygame**.

The game features two playable levels, local co-op support for two players, and a local ranking system powered by **SQLite3**. Each level ends when its time limit is reached, challenging players to achieve the highest score possible before time runs out.

## Project Origin

This project was developed as an academic assignment for the **Applied Programming Logic** course offered by **Uninter University** and proposed by **Professor Vinicius Pozzobon Borin**.
